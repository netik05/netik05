## 🐍 Contribution Snake

  <img src="https://raw.githubusercontent.com/netik05/netik05/output/github-contribution-grid-snake.svg" alt="Contribution Snake Animation" width="100%" />

<details>
<summary>🔧 <b>Setup the Snake Animation</b> (click to expand)</summary>

<br/>

This animation is generated by a GitHub Action that runs daily. To enable it:

1. In your `netik05/netik05` profile repo, create a workflow file at `.github/workflows/snake.yml`
2. Paste the following:

```yaml
name: Generate Snake

on:
  schedule:
    - cron: "0 0 * * *"
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - name: Generate Snake
        uses: Platane/snk/svg-only@v3
        with:
          github_user_name: netik05
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark

      - name: Push to output branch
        uses: crazy-max/ghaction-github-pages@v3.1.1
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

3. Go to **Actions** tab → enable workflows → manually trigger "Generate Snake" once
4. The animation will appear after the first run completes

</details>
